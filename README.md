# PageAudit — Deploy to GitHub Pages (Free, 5 minutes)

## You will get a permanent URL like:
`https://YOUR-USERNAME.github.io/pageaudit/`

Share that link with your team — it works in any browser, no install needed.

---

## Step-by-step

### 1. Create a GitHub account (if you don't have one)
Go to https://github.com/signup — it's free.

### 2. Create a new repository
- Go to https://github.com/new
- Repository name: `pageaudit`
- Set to **Public**
- Click **Create repository**

### 3. Upload the file
- On your new repo page, click **"uploading an existing file"**
- Drag `index.html` into the upload area
- Click **Commit changes**

### 4. Enable GitHub Pages
- Go to your repo → **Settings** tab
- Left sidebar → **Pages**
- Under "Branch", select `main` → `/ (root)` → click **Save**

### 5. Your URL is live in ~60 seconds
- GitHub will show: `Your site is live at https://YOUR-USERNAME.github.io/pageaudit/`
- Share that URL with your team

- Works in any browser
